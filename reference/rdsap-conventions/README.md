# RdSAP Conventions — Reference Library

This folder contains the RdSAP 10 conventions and technical documentation.
It is used by:
1. The content engine — as primary source when generating course content
2. The course library — made available to subscribers as a reference resource
3. NotebookLM — uploaded as a source for research queries

## What Belongs Here

- RdSAP 10 Technical Manual (PDF when obtained)
- RdSAP 10 Conventions Document (PDF when obtained)
- RdSAP 10 Change Log / What's New summary
- Any BRE/DESNZ technical notes on RdSAP 10 implementation
- Previous version (RdSAP 9.94) for comparison reference

## How to Get These Documents

The official RdSAP 10 documentation is published by BRE on behalf of DESNZ.
Sources:
- bregroup.com/products/sap — SAP/RdSAP technical documentation
- gov.uk/guidance/standard-assessment-procedure — official SAP page
- Elmhurst, Quidos, ECMK — accreditation body technical bulletins on RdSAP 10
